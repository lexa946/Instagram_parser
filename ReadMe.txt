Для того, чтобы начать работать с программой - нужно:
1.Установить Python 3 (самую последнюю сборку на оффициальном сайте)
2.При установке поставить галочки:
	a)Установить для всех пользователей
	b)Добавить в переменную PATH
3.Установить дополнительные библиотеки PyQt5, InstagramAPI
(В командной строке вписать: pip install pyqt5, pip install instagram api;
Если не получается попробовать командную строку открыть от Администратора)
4.Открываем pyqt_test 
5.Вводим логин\пароль от инстаграмма
*ВАЖНО: Если программа вылетает, скачивайте на телефон приложение Device ID,
и введите ID вашего телефона в поле "Device ID"
6.Выбирайте базу, устанавливайте значения и PROFIT!
Базу желательно хранить в директории с программой
(тестовую базу подписчиков так же прилагаю)
